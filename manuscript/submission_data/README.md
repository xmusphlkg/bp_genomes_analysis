# Submission Data

This directory stores submission-facing data artifacts that are consumed by the manuscript package but are not themselves prose.

## Layout

- `cohort/`: cohort-contract TSVs such as the sample-level decision log and stage-level flow summary.
- `validation/`: manuscript-facing validation ledgers.
- `source_data/`: source-data manifests and figure-specific Excel workbooks.

## Rule

- Narrative explanation stays in `docs/`.
- Data contracts, ledgers, manifests, and workbooks stay under `manuscript/submission_data/`.
