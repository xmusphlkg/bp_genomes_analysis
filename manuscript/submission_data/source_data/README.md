# Source Data

This directory contains one Excel workbook per figure for the submission package.
Each workbook contains one sheet per panel, with source-file headers followed by the exported table blocks used to build that panel.

`Base manifest`: `manuscript/submission_data/source_data/final_source_data_manifest.tsv`
`Panel manifest`: `manuscript/submission_data/source_data/final_source_data_panel_manifest.tsv`
`File manifest`: `manuscript/submission_data/source_data/final_source_data_file_manifest.tsv`
