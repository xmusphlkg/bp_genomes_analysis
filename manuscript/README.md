# Manuscript Reproducibility Package

This directory contains the public, manuscript-facing reproducibility package for
the *Bordetella pertussis* genome-defined *prn* disruption study.

It is organized for three uses:

- Inspecting the frozen numerical values behind the main and supplementary figures.
- Reusing Supplementary Tables and Source Data workbooks.
- Rebuilding manuscript-facing figure data and rendered figures from tracked
  workflow outputs where those upstream outputs are available.

Private submission text, cover letters, author metadata, reporting-summary notes,
editorial checklist notes, and internal review scripts are intentionally excluded
from the public release.

## Public Contents

- `PUBLIC_RELEASE.md`: manuscript-level public release boundary.
- `figure_data/`: plot-ready TSV extracts and analysis-facing ledgers.
- `figure_data_dictionary.md`: data dictionary for the figure-data layer.
- `figures/README.md`: figure rendering notes.
- `figures/scripts/`: R scripts used to render main and supplementary figures.
- `figures/bin/`: figure rendering entry points.
- `figures/outputs/main/`: rendered main figure PDFs and PNGs.
- `figures/outputs/extended_data/`: rendered supplementary-figure source images.
- `supplementary/`: machine-readable Supplementary Tables.
- `submission_data/`: source-data workbooks, cohort logs, and validation ledgers.
- `curation/selected_country/`: curated overrides used by selected-country audits.
- `key_statistics.json` and `key_statistics.txt`: frozen key statistics extracts.
- `bin/` and selected `scripts/`: refresh, freeze, sidecar, diagnostic, and
  source-data builders used for manuscript-facing outputs.

## Attachment Naming Note

Files under `figures/outputs/extended_data/` retain internal
`Extended_Data_Fig_*` filenames because they are generated by the figure renderer.
In the Communications Biology submission text, these correspond to Supplementary
Figures 1-15. Source-data workbooks use `Source_Data_Supplementary_Figure_*`
filenames for supplementary figures. Supplementary Tables 1-12 are the official
reader-facing TSV tables; the historical wider table ledgers are retained under
`submission_data/audit_ledgers/` as reproducibility inputs rather than numbered
Supplementary Tables.

## Refresh Entry Points

```bash
bash manuscript/bin/refresh_submission.sh
Rscript manuscript/figures/bin/render_main.R
Rscript manuscript/figures/bin/render_extended_data.R
```

`bin/refresh_submission.sh` regenerates figure data, supplementary tables, and
source-data workbooks from upstream workflow outputs. The figure renderers create
the current main and supplementary figure assets from the frozen figure-data layer.
