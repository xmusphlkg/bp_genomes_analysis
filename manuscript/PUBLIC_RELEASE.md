# Manuscript Public Release Boundary

This file defines the manuscript-level subset that can be extracted for a public reproducibility package. It intentionally excludes private submission text, cover letters, checklist notes, author metadata and internal audit files.

## Public Reader/Reproducibility Package

Include these paths when preparing a lightweight public package:

- `README.md`
- `PUBLIC_RELEASE.md`
- `figure_data/`
- `figure_data_dictionary.md`
- `figures/README.md`
- `figures/outputs/main/`
- `figures/outputs/extended_data/`
- `figures/scripts/`
- `figures/bin/`
- `supplementary/`
- `submission_data/README.md`
- `submission_data/cohort/`
- `submission_data/source_data/`
- `submission_data/validation/`
- `curation/selected_country/`
- `key_statistics.json`
- `key_statistics.txt`

## Optional Code-Level Refresh Layer

Include these only if the public package should support rebuilding frozen manuscript-facing extracts from the workflow outputs:

- `bin/refresh_submission.sh`
- `bin/build_biology_bridge_sidecars.py`
- `scripts/freeze/`
- `scripts/sidecars/`
- `scripts/diagnostics/`
- `scripts/source_data/`

These scripts assume the wider project workflow, configured environments and upstream output files are available. They are useful for a code release, but they are not required for reader-level inspection of figures, supplementary tables and source-data workbooks.

## Internal-Only Paths

Do not include these in the public package:

- `text/`
- `text/templates/`
- `scripts/review/`
- `scripts/validate_commsbio_submission.py`
- `focal_country_dynamics_audit.md`
- any `__pycache__/`, `_archive/`, temporary render products, local editor files or generated submission PDFs/Word files

## Attachment Naming Note

Files under `figures/outputs/extended_data/` retain internal `Extended_Data_Fig_*` names because they are generated by the figure renderer. In the Communications Biology submission text, these correspond to Supplementary Figures 1-15. Source-data workbooks use `Source_Data_Supplementary_Figure_*` filenames for supplementary figures. Supplementary Tables 1-12 are the official reader-facing TSV tables; the historical wider table ledgers are retained under `submission_data/audit_ledgers/` as reproducibility inputs rather than numbered Supplementary Tables.
